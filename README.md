## ✨ Key Features

This is not a standard template. It was built from scratch with a focus on premium aesthetics and developer-first features:

- **Bento Grid Architecture**: A modern, interlocking CSS Grid layout showcasing 8+ backend and ML projects.
- **Interactive Terminal Emulator**: A fully functional JavaScript terminal in the hero section. Try typing `help`, `whoami`, `skills`, or `projects`!
- **Glassmorphism & Neon Orbs**: Custom CSS styling utilizing `backdrop-filter` and animated CSS gradients to create a deep, immersive dark mode.
- **Live GitHub Heatmap**: Real-time integration of my GitHub commit history directly into the capabilities section to prove daily consistency.
- **Custom Python Static Site Generator**: The engineering blog is entirely statically generated using a custom Python script (`scripts/build_blog.py`) that parses Markdown files into perfectly styled HTML.
- **Premium Stylography**: Developer-focused font pairing using **Plus Jakarta Sans** (editorial) and **Fira Code** (monospace), complete with custom scrollbars and text-selection highlights.

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Typography**: Plus Jakarta Sans, Fira Code
- **Blog Engine**: Python 3 (Markdown parsing)
- **Deployment**: GitHub Pages
- **Version Control**: Git

## 📝 Running the Blog Generator

To add a new blog post to the site:
1. Navigate to the `posts/` directory and create a new markdown file (e.g., `my-new-post.md`).
2. Run the custom Python build script from the root directory:

```bash
# Make sure you have python installed
python3 scripts/build_blog.py
```

3. The script will automatically parse your Markdown, wrap it in the premium HTML template, generate a new page in `blog/`, and update the `blog.html` index list!
